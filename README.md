# Proyecto-Cinemax
Proyecto Cinemax, Cine Backend con 10 Microservicios, ms-peliculas, ms-sucursales, ms-catalogo, ms-usuario, ms-butacas, ms-reserva, ms-pago, ms-confiteria, ms-ticket, ms-notificacion. Todos este programa fue realizado mediante Spring Boot, Lombok, Laragon y HeidiSQL para las respectivas bases de datos, programado con el lenguaje Java.
